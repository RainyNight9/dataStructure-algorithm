# 堆排序

- 一种基于完全二叉树的结构
- 大顶堆和小顶堆
  - 大顶堆 —— 任意的三元组，父节点都大于两个子节点
  - 小顶堆 —— 任意的三元组，父节点都小于两个子节点
  - 根节点为最大值或最小值
- 堆适合维护集合的最值

堆排序是一种很高效的算法，因其把数组当作二叉树来排序而得名。

通过将待排序的元素构建成一个最大堆（或最小堆），然后逐步将堆顶元素与堆尾元素交换，并重新调整堆，使得堆的大小减小，直到堆中仅剩一个元素，最后完成排序。

## 堆的基本操作（以大顶堆为例）

- 尾部插入
  - 比父节点大就和父节点交换 递归向上调整
  - 这个过程称为SIFT-UP
- 头部弹出
  - 用最后一个元素（叶子结点）补位 递归向下调整
  - 这个过程称为SIFT-DOWN
